# 🛡️ SchemeMitra AI Engine

An enterprise-level Relational Scheme Discovery Engine developed using **Streamlit** and powered by local **Llama 3 (via Ollama)** and **PostgreSQL**.

## 🚀 Features
- **Natural Language to SQL:** Converts user queries in plain text into precise SQL database commands.
- **Dynamic Policy Matching:** Instantly filters eligible welfare schemes based on age, caste, and family income.
- **Interactive UI:** Built using a custom premium dark-themed Streamlit dashboard.

## 🛠️ Tech Stack
- **Frontend:** Streamlit
- **LLM Engine:** Llama 3 (Ollama)
- **Database:** PostgreSQL / Python-psycopg2

## 📦 Installation & Setup
1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `streamlit run app.py`
