Phase 6 Backend

Local Server & Ngrok Tunnel Setup

  • 1

    Flask Environment: Command prompt par Flask dependencies install karein:
    pip install flask requests

  • 2

    Ngrok Configuration: Ngrok account par signup karke authtoken link configure karein:
    ngrok config add-authtoken <YOUR_TOKEN>

  • 3

    Start Tunneling: Port 5000 par secure HTTPS URL tunnel start karein:
    ngrok http 5000
